# Ирина Буцкая — Лендинг эксперта по управлению качеством

Одностраничный сайт + набор генераторов документов по качеству (FMEA, Control Plan, PSW, APQP, PPAP, SPC, MSA и др.).

## Структура

```
butskaya-landing/
├── landing/          # лендинг (index.html + favicon)
├── shop/             # магазин PRO-шаблонов PDF
├── generators/       # онлайн-генераторы документов качества
└── docs/             # ТЗ, черновики, блок-схемы
```

## Публикация

1. Загрузить всё содержимое папки на хостинг или GitHub Pages
2. Лендинг откроется по `landing/index.html`
3. Генераторы — по `generators/index.html`
