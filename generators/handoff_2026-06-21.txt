## О чём был чат

Сессия по созданию сайта на основе 39 готовых HTML-страниц по инструментам качества (Core Tools: APQP, PPAP, FMEA, MSA, SPC; 7 QC Tools; Kaizen; 5S). Идея: пользователь заполняет формы → получает готовый PDF с данными своей компании.

- **Привязка:** с лендинга ведут ссылки на fmea-generator.html, controlplan.html, psw.html, index.html. Обратно — ссылки «← Назад к Ирине Буцкой» на fmea-generator.html, index.html, companyinfo.html.

## Ключевые решения и почему

- **Язык: русский для интерфейса** — пользователь выбрал вариант Б (полностью на русском). Сами профессиональные формы (controlplan.html, psw.html) остались на английском — это стандарт IATF.
- **Paywall через модалку** — самый быстрый способ поставить заглушку без серверной части. При нажатии Export PDF — всплывает окно с предложением оплаты и контактом (@Irina_Butskaya).
- **Модель монетизации:** бесплатно (PSW, Company Info, QA vs QC) / подписка $12/мес / pay-per-PDF $5.
- **Пути:** использованы относительные через `../OneDrive/Desktop/Сайт/сайт по качеству/` и `../../../Documents/лендинг_Ирина_Буцкая.html`.
- **FMEA Generator сделан новым целиком** — старый FMEA_Premium_English.html оставлен как справочный материал.

## Новая цель: сайт для тестирования соответствия ISO 9001

Пользователь хочет создать сайт, где компания проходит **онлайн-опрос / чеклист**, и система оценивает, насколько её процессы соответствуют требованиям ISO 9001.

## План работ (следующие шаги)

1. **Разработать чеклист соответствия ISO 9001** — список вопросов по разделам стандарта (контекст организации, лидерство, планирование, обеспечение, производство, оценка, улучшение)
2. **Создать HTML-форму** с вопросами, вариантами ответов (да/нет/частично/не применимо), подсчётом процента соответствия
3. **Сгенерировать PDF-отчёт** с результатами — сводка по разделам, общий %, рекомендации
4. **Интегрировать в лендинг** — добавить как новый инструмент/услугу
5. Оставить ту же модель монетизации: бесплатный чеклист / платный подробный PDF-отчёт
